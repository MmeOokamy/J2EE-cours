L'objectif est de créer une application autour de la base geo, en utilisant les notions suivantes :
•Servlet.
•Page JSP.
•Persistance des données avec JPA.
•Architecture MVC.
•HTML / CSS + javascript.

Guide.

Créer un nouveau siteweb dynamique.
•Référencer les librairies nécessaires: connecteur pour MySql, JPA, etc.
•Ajouter une page index.htm(peu utile dans ce projet).
•Ajouter une page region-view.jsp sur ce modèle : imgage pdf

•Ajouter un écouteur pour les événements de ContextServlet.

Coder ici la connexion à la base de données.

•Ajouter une servlet RegionController.Coder ici :
-Le remplissage de la liste de choix d'une région.
-La recherche d'une région à partir de la liste de choix d'une région.
-Le bouton Ajouter (effacement du formulaire).
-Le bouton supprimer.
-Le bouton Enregistrer.
